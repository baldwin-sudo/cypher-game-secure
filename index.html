Welcome to the Deciphering Game

Game Explanation

In this challenge, you need to decrypt a secret message using a Caesar Cipher and solve an anagram. Each word has been right-shifted by a key, which is the length of the word itself. Can you uncover the final phrase?

Each participant is allowed one hint or answer to assist them during the challenge. Use it wisely!

Here is the encrypted phrase: "DLSJTVL VQ KFYFQ JWWTW"

  • Caesar Cipher: Each word is scrambled by shifting the letters based on the length of the word.
  • Anagram: After decrypting the words, rearrange the letters to solve the final anagram.

Algorithm Explanations

A Caesar cipher shifts each letter in a message by a set number. The key here is the length of each word. For example, if the word "LOOK" has 4 letters, each letter will be shifted by 4 positions to get "HKKG".

An anagram rearranges the letters of a word or phrase into a new order. Once you decrypt the scrambled words using the Caesar cipher, you'll need to solve the anagram to reveal the final message.

Solution Explanation

To solve the cipher, follow these steps:

  1. Decrypt the Caesar Cipher: Shift the letters in each word back by the number equal to the length of the word.
  2. Solve the Anagram: Rearrange the letters of the decrypted words to find the final hidden message.

By following these steps, you will reveal the hidden message and complete the challenge!
